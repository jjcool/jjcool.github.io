=== Parameter Set

This section is used when ratio variables are to be generated.
To generate ratio variables you need to specify parameter sets for the data source on which you want to calculate the ratio variables. Parameter sets configurations are like variable outlines, except they need a parameter name for ex. mysetA and variable function is not specified in the  Parameter Set
. It is specified in the <<ratio-variable-config,Ratio config>>
